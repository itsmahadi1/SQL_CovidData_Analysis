# COVID-19 SQL Data Exploration

## Overview
This project explores global COVID-19 data using SQL Server.
The analysis focuses on infection rates, death percentages,
and vaccination progress using analytical SQL techniques.

## Data Preprocessing
I have divided the datset into two files
CovidDeaths.xlsx file contains information about new cases and deaths.
CovidVaccinations.xlsx file contains information about vaccines.

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)
- T-SQL

## Key SQL Concepts Demonstrated
- JOINs
- Window functions
- Rolling aggregates
- TRY_CAST for safe data conversion
- Data cleaning
- Analytical queries

## Dataset
Source: Our World in Data (COVID-19 dataset)

## Notes
This project is based on a common SQL learning walkthrough and
was refactored to follow professional SQL Server best practices.
